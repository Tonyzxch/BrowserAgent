## 🎯 测试步骤

```
1. 打开新标签页
2. 访问 https://www.baidu.com
3. 刷新页面（F5）⚠️ 重要！
```

### Step 3: 测试基本功能
```
1. 点击插件图标
2. 应该看到紫色渐变的弹窗
3. 输入测试任务："在百度搜索人工智能"
4. 点击"开始执行"
5. 侧边栏应该自动打开
```

### Step 4: 观察执行过程
```
侧边栏应该显示：
- 任务状态：规划中 → 执行中 → 已完成
- 步骤列表：每个步骤的执行情况
- 元素高亮：操作的元素会有红色边框
```

## 🐛 故障排查

### 如果出现"Could not establish connection"
1. ✅ 刷新测试网页（F5）
2. ✅ 重新加载插件
3. ✅ 查看Console是否有错误

### 如果AI服务连接失败
1. 检查网络连接
2. 验证API Key是否正确
3. 查看Background日志

### 如果元素定位失败
1. 页面可能还在加载，等待几秒
2. 尝试更简单的任务
3. 查看Content Script日志
