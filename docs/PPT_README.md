## 自定义修改

如果需要修改内容，可以：

1. **直接编辑PPT文件**（推荐）
   - 用PowerPoint或WPS打开 `docs/BrowserAgent_Presentation.pptx`
   - 修改文字、调整布局、添加图片

2. **修改生成脚本**
   - 编辑 `scripts/generate-presentation.py`
   - 修改 `add_content_slide()` 函数中的内容
   - 重新运行：`python scripts/generate-presentation.py`

## 添加个人信息

如果需要在PPT中添加你的个人信息：

1. 打开 `docs/BrowserAgent_Presentation.pptx`
2. 在第10页（结束页）的"联系方式：见个人简历"处
3. 替换为你的实际联系方式：
   - 📧 Email: your.email@example.com
   - 💼 LinkedIn: linkedin.com/in/yourprofile
   - 🐙 GitHub: github.com/yourusername

或者在第1页（封面）添加作者信息：
   - 在副标题下方添加："作者：Xucheng Zhuang"
