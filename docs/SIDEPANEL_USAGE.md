# 使用说明更新

## 侧边栏使用方式

由于Chrome Extension的限制，侧边栏无法自动打开。请按照以下方式使用：

### 方法1：手动打开侧边栏（推荐）

1. 创建任务后，popup会自动关闭
2. **点击Chrome工具栏的侧边栏图标** 📌
3. 或者右键点击插件图标 → "打开侧边栏"
4. 在侧边栏中查看任务执行进度

### 方法2：使用快捷键

- Windows/Linux: `Ctrl + Shift + E`
- Mac: `Cmd + Shift + E`

## 工作流程

```
1. 点击插件图标 → 打开Popup
2. 输入任务 → 点击"开始执行"
3. Popup自动关闭
4. 手动打开侧边栏 → 查看任务进度
```

## 注意事项

⚠️ **侧边栏不会自动打开** - 这是Chrome的安全限制，需要用户主动操作

✅ **任务会在后台执行** - 即使没有打开侧边栏，任务也会正常执行

✅ **可以随时打开侧边栏** - 任务执行过程中随时可以打开侧边栏查看进度
