# 浏览器智能助手 - 演示脚本

本文档提供完整的演示流程，帮助你录制项目演示视频。

## 演示准备

### 环境检查
- [ ] Chrome浏览器已安装插件
- [ ] 插件可以正常工作
- [ ] AI服务连接正常
- [ ] 录屏软件已准备（OBS/Camtasia等）

### 测试网站准备
建议使用以下网站进行演示：
1. GitHub (搜索项目)
2. 豆瓣电影 (信息查询)
3. 百度 (简单搜索)

## 演示脚本

### 第一部分：项目介绍（30秒）

**旁白**：
"大家好，这是浏览器智能助手项目。这是一个基于AI的Chrome扩展，可以通过自然语言控制浏览器自动完成复杂任务。"

**画面**：
- 展示Chrome工具栏中的插件图标
- 简单点击打开popup界面
- 快速浏览界面元素

### 第二部分：功能演示1 - GitHub搜索（1分钟）

**场景**：在GitHub上搜索React UI组件库

**操作步骤**：
1. 打开 https://github.com
2. 点击插件图标
3. 输入任务："帮我搜索React相关的UI组件库，按star排序"
4. 点击"开始执行"
5. 侧边栏自动打开
6. 展示任务规划步骤
7. 观看AI自动操作：
   - 定位搜索框
   - 输入"React UI library"
   - 点击搜索
   - 应用排序筛选

**旁白**：
"现在我们让AI帮我们在GitHub搜索React相关的UI组件库。我用自然语言描述任务，AI会自动理解意图、分析页面结构、规划执行步骤，然后自动操作浏览器完成任务。"

### 第三部分：功能演示2 - 信息提取（1分钟）

**场景**：在豆瓣电影查找高分科幻片

**操作步骤**：
1. 打开 https://movie.douban.com
2. 点击插件图标
3. 输入任务："搜索评分8分以上的科幻电影"
4. 观看执行过程：
   - 导航到搜索页面
   - 输入搜索关键词
   - 应用评分筛选
   - 提取电影信息

**旁白**：
"接下来演示信息提取功能。我让AI帮我找豆瓣上的高分科幻电影。AI会自动搜索、筛选、并整理信息。"

### 第四部分：技术架构（30秒）

**画面**：
- 展示项目文件结构
- 简要说明核心模块
- 展示代码片段（可选）

**旁白**：
"项目采用React + TypeScript开发，使用Chrome Extension Manifest V3。核心包括页面感知模块、AI规划引擎和动作执行器。支持Docker部署，完整代码已开源。"

### 第五部分：设计亮点（30秒）

**旁白重点**：
1. **智能页面感知**：自动提取页面可交互元素，生成精准选择器

3. **可视化反馈**：实时展示任务进度，高亮操作元素
4. **灵活扩展**：模块化设计，易于添加新动作类型

**画面**：
- 展示侧边栏实时更新
- 展示元素高亮效果

### 第六部分：总结（20秒）

**旁白**：
"浏览器智能助手让复杂的网页操作变得简单。只需用自然语言描述任务，AI就能帮你自动完成。项目已开源，欢迎试用和贡献。"

## 演示技巧

### 录屏设置
- 分辨率：1920x1080
- 帧率：30fps
- 录制区域：全屏或浏览器窗口
- 音频：清晰的旁白

### 注意事项
1. **提前测试**：确保所有功能正常工作
2. **语速适中**：不要说太快，让观众有时间理解
3. **画面清晰**：确保文字和操作清晰可见
4. **突出重点**：使用箭头或高亮标注关键信息
5. **时间控制**：总时长控制在3-5分钟内

### 可选增强
- 添加背景音乐（轻快、不抢戏）
- 添加字幕（中英文）
- 添加转场效果
- 添加代码展示片段
- 添加架构图示

## 应急方案

### 如果演示失败
1. **准备录好的备用片段**
2. **准备截图说明**
3. **提前录制成功案例**
